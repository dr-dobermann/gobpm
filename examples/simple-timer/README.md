# Simple Timer Example

A minimal example focusing on timer functionality with the simplest possible implementation.

## What it demonstrates

- Minimal timer setup with timeDate
- Basic timer start event
- Minimal process structure
- Simplified timer implementation

## Key concepts

- Minimal timer implementation
- Timer start events
- Basic process flow
- Essential timer patterns

## Running the example

```bash
go run main.go
```

## Expected output

```
Timer process started. Will trigger in 3 seconds...
Timer should have fired by now!
```

## Code walkthrough

1. **Simple Setup**: Minimal engine and process creation
2. **Timer Expression**: Basic timeDate expression for 3 seconds from now
3. **Timer Event**: Start event with timer trigger
4. **Direct Flow**: Timer start directly to end event
5. **Basic Execution**: Simple process execution with timer

## When to use

This example is perfect for:
- Learning basic timer concepts
- Understanding minimal timer setup
- Quick prototyping of timer-based processes
- Educational purposes

Start with this example if you're new to timer events in BPMN.
